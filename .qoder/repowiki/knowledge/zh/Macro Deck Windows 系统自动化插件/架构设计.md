- **插件入口与注册**：`Main.cs` 继承自 `MacroDeckPlugin`，在 `Enable()` 中实例化并注册所有 `PluginAction` 子类（如 `HotkeyAction`, `StartApplicationAction`）。
- **动作执行层 (Actions)**：包含多个继承自 `PluginAction` 的具体动作类，通过 `Trigger` 方法执行逻辑。配置数据通常以 JSON 字符串形式存储在 `Configuration` 属性中。
- **配置交互层 (GUI/Views/ViewModels)**：采用 MVVM 模式处理复杂配置。`Views` 定义 UI 界面，`ViewModels` 处理数据绑定与序列化逻辑（实现 `ISerializableConfigViewModel`），`GUI` 文件夹下包含传统的 WinForms 配置控件。
- **核心服务与工具 (Services/Utils)**：`ApplicationLauncher` 封装了 P/Invoke 调用以管理进程窗口状态；`Utils` 提供图标提取、快捷方式解析等底层支持。
- **多语言支持 (Language)**：`PluginLanguageManager` 从嵌入的 XML 资源中动态加载本地化字符串，支持运行时语言切换。
